# Microsoft Store Description – FileStrider

## English (en-US)

### Short Description
Find the biggest files and folders on your disk in seconds.

### Description
FileStrider is a fast, free disk space analyzer that helps you quickly discover what's eating up your storage.

**Key Features:**
• Fast scanning — efficiently scans directories to find the largest files and folders
• File type analysis — see disk usage broken down by category (images, videos, documents, archives, code, and more)
• Top results — configurable list of the biggest items (up to 200)
• Detailed info — file and folder sizes, modification dates, and item counts
• Double-click to open — jump straight to any file or folder in File Explorer
• Export results — save your scan to CSV or JSON, including file type statistics
• Flexible filters — include/exclude hidden files, set minimum size, exclude directories like node_modules or .git
• Real-time progress — live updates as the scan runs
• Cancellable — stop any scan at any time
• Modern UI — clean three-column layout with treemap visualization
• Multi-language — English, French, Spanish, and Swedish

FileStrider is open source and collects no data. Your files stay on your device.

### Features
- Fast file system scanning
- File type analysis with visual breakdown
- Export to CSV and JSON
- Double-click to open in File Explorer
- Configurable filters and scan depth
- Real-time progress with cancellation
- Treemap visualization
- Multi-language support (EN, FR, ES, SV)

### Keywords
disk space, storage analyzer, large files, folder size, disk usage, file manager, cleanup, storage management

## Swedish (sv-SE)

### Short Description
Hitta de största filerna och mapparna på din disk på några sekunder.

### Description
FileStrider är en snabb och gratis diskutrymmesanalysator som hjälper dig att snabbt upptäcka vad som tar upp ditt lagringsutrymme.

**Huvudfunktioner:**
• Snabb skanning — skannar kataloger effektivt för att hitta de största filerna och mapparna
• Filtypanalys — se diskanvändning uppdelad efter kategori (bilder, videor, dokument, arkiv, kod med mera)
• Toppresultat — konfigurerbar lista över de största objekten (upp till 200)
• Detaljerad information — fil- och mappstorlekar, ändringsdatum och antal objekt
• Dubbelklicka för att öppna — hoppa direkt till valfri fil eller mapp i Utforskaren
• Exportera resultat — spara din skanning som CSV eller JSON, inklusive filtypstatistik
• Flexibla filter — inkludera/exkludera dolda filer, ange minimistorlek, exkludera kataloger som node_modules eller .git
• Realtidsförlopp — liveuppdateringar medan skanningen körs
• Avbrytbar — stoppa vilken skanning som helst när som helst
• Modernt gränssnitt — ren trespaltig layout med trädkartsvisualisering
• Flerspråkig — engelska, franska, spanska och svenska

FileStrider är öppen källkod och samlar inte in några data. Dina filer stannar på din enhet.

### Features
- Snabb filsystemskanning
- Filtypanalys med visuell uppdelning
- Export till CSV och JSON
- Dubbelklicka för att öppna i Utforskaren
- Konfigurerbara filter och skanningsdjup
- Realtidsförlopp med avbrytning
- Trädkartsvisualisering
- Flerspråkigt stöd (EN, FR, ES, SV)
